# CreativeMinds Frontend Assessment

Two independent submissions, each in its own folder with its own README:

- **[`market-dashboard/`](market-dashboard/README.md)** — Real-time Market Dashboard: Binance
  REST + WebSocket integration, Redux Toolkit, connection status, reconnection.
- **[`figma-dashboard/`](figma-dashboard/README.md)** — Figma UI recreation of the provided
  project dashboard design.

Each app is a standalone Vite + React + TypeScript project with its own `npm install` / `npm run dev`.
